# Welcome to Awesome ML Courses

This documentation site guides you through the structure and usage of the Awesome ML Courses platform.

## Getting Started

- **[Explore Courses](/explore)** - Browse the full catalog of curated ML courses
- **[Learning Paths](/learning-paths)** - Follow structured paths to learn ML systematically
- **[About](/about)** - Learn more about this project

## What is Awesome ML Courses?

Awesome ML Courses is a carefully curated collection of the best machine learning and artificial intelligence courses from top universities worldwide. Our platform provides:

- **Comprehensive Course Database** - Hundreds of high-quality courses from Stanford, MIT, CMU, and online platforms
- **Intelligent Search & Filtering** - Find courses by university, difficulty level, and specialization
- **Learning Paths** - Pre-designed sequences of courses that build upon each other
- **Rich Metadata** - Details about video availability, solutions, and course difficulty

## Course Categories

Our collection covers:

- **Fundamentals** - Linear algebra, calculus, probability and statistics
- **Core ML** - Supervised learning, unsupervised learning, deep learning
- **Specialized Topics** - NLP, computer vision, reinforcement learning, generative AI
- **Production ML** - MLOps, model deployment, system design
- **Advanced Topics** - Research papers, cutting-edge techniques, specialized architectures

## How to Use

### Search and Filter

Use the search bar to find courses by name, topic, or university. Apply filters to narrow results by:
- **Section** - Categorized learning areas
- **University** - Specific institution or platform
- **Type** - Course, textbook, or other resource
- **Starred** - Community favorites

### Follow Learning Paths

Choose a learning path that matches your goals and skill level. Each path provides a logical sequence of courses to build comprehensive knowledge.

### External Resources

Each course card links to the original course page for enrollment and access information.
