# Contributing to Awesome Full-Stack Machine Learning Courses

First and foremost, thank you for considering contributing to this project! We're excited to have you help curate a collection of high-quality ML/AI courses. Your contributions make this resource more valuable for everyone learning about machine learning engineering.

## Guidelines for Suggesting a New Course

Before submitting a course, please ensure it meets the following criteria:

### Course Requirements

- **Reputable Source**: The course must come from an established university, recognized online platform (Coursera, edX, MIT OpenCourseWare, etc.), or well-known educational institution.
- **Publicly Accessible**: Course materials must be freely available and remain accessible. This includes video lectures, assignments, notes, or other learning resources. Paywalled or heavily restricted content should not be included.
- **ML Engineering Relevance**: The course should directly relate to machine learning engineering, not just tangentially touch on ML. We focus on practical, hands-on learning that builds real ML systems.
- **Python Preference**: Courses using Python are preferred, as it's the dominant language in ML engineering. However, content in other languages may be considered if exceptionally high quality.

### Submission Format

Please use the following format for course entries:

```
- [Course Name](URL) - Brief description (1-2 sentences) covering key topics and learning outcomes.
```

**Optional**: Exceptional courses can be marked with a :star: emoji:

```
- :star: [Course Name](URL) - Brief description of why this course stands out.
```

## How to Submit

### Step-by-Step Process

1. **Fork the Repository**: Click the fork button to create your own copy of the repo.
2. **Edit README.md**: Make your changes directly in the README.md file—it's the single source of truth for course listings. Do not create separate files.
3. **Submit a Pull Request**:
   - Write a clear, descriptive PR title: e.g., "Add [Course Name] to [Section]"
   - In the PR description, explain:
     - What course(s) you're adding
     - Why you believe it belongs in this collection
     - Any relevant notes about the course quality or accessibility
   - **One course per PR is preferred** to keep reviews focused and manageable.

## Quality Standards

Before submitting your PR, please verify:

- [ ] All links are active and not broken
- [ ] The course materials are currently publicly accessible
- [ ] The course entry is placed in the correct section of README.md
- [ ] Formatting exactly matches existing entries (spacing, punctuation, link syntax)
- [ ] The description is clear, concise, and accurately represents the course content
- [ ] If using a :star:, the course truly is exceptional (widely recognized, comprehensive, exceptionally well-taught)

## Reporting Issues

Found a problem? We appreciate it! Please open an issue for:

- **Broken Links**: Courses that no longer work or have moved
- **Outdated Information**: Courses that are no longer publicly accessible or have changed significantly
- **New Section Suggestions**: Ideas for organizing courses differently or adding new categories
- **Other Concerns**: Any other quality or content issues

Include as much detail as possible so we can investigate and fix the issue quickly.

## Code of Conduct

We're committed to fostering an inclusive, welcoming community. When contributing, please:

- **Be respectful** and constructive in all communications
- **Assume good intent** when reviewing or discussing submissions
- **Provide helpful feedback** focused on content quality, not personal criticism
- **Welcome diverse perspectives** and learning preferences

Let's build an amazing resource together!
